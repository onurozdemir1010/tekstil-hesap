# Tekstil Hesap PWA Kurulumu

1. Supabase'de yeni proje açın.
2. `supabase-schema.sql` dosyasındaki komutları Supabase SQL Editor içinde çalıştırın.
3. Supabase Project Settings > API ekranından `Project URL` ve `anon public` key bilgisini alın.
4. Uygulamayı açın, `Ayarlar` sekmesine girin, URL ve anon key bilgisini kaydedin.
5. Eski cihazdaki yerel kayıtları taşımak için `Yereli Buluta Gönder` düğmesini kullanın.

Not: Bu ilk ekip sürümü küçük ve kapalı ekip kullanımı için pratik olacak şekilde yazıldı. Daha sıkı kullanıcı girişleri istenirse Supabase Auth ve kullanıcı bazlı yetkiler eklenebilir.
